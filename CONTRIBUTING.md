# Contributing to FUI-RS

This file is for people working on the SDK itself. Application developers
should start with the [FUI-RS developer quickstart](QUICKSTART.md).

Contributor prerequisites, build commands, test lanes, and architecture rules
are maintained in the
[FUI-RS contributor quickstart](docs/v2/fui-rs/CONTRIBUTOR_QUICKSTART.md).

## Getting in touch

This is a solo project. If you're thinking about contributing, please open an
issue or start a discussion before writing code.
